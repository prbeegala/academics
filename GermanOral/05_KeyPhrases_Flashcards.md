# 05 — Schlüsselphrasen (Master Flashcards)

Drill these until they're automatic. They work across **every question**.

---

## 🌟 Top 10 phrases that lift any answer

| German | English |
|---|---|
| Meiner Meinung nach … | In my opinion … |
| Ich finde, dass … | I find / think that … |
| Es kommt darauf an. | It depends. |
| Einerseits … andererseits … | On the one hand … on the other hand … |
| Zum Beispiel … | For example … |
| Außerdem … | Furthermore … |
| Im Allgemeinen … | In general … |
| Das macht (mir) Spaß. | I enjoy that. |
| Es gefällt mir (sehr / gar nicht). | I like it (very much / not at all). |
| Ich interessiere mich für … | I'm interested in … |

---

## 🔗 Connectives (= automatic marks!)

### Coordinating (verb in normal position)
- **und** — and
- **aber** — but
- **oder** — or
- **denn** — because (informal)
- **sondern** — but rather

### Subordinating (verb to the END!)
- **weil** — because  → *Ich mag Tennis, **weil** es Spaß **macht**.*
- **dass** — that  → *Ich finde, **dass** Sport gesund **ist**.*
- **obwohl** — although  → *…, **obwohl** es schwierig **ist**.*
- **wenn** — when / if  → *…, **wenn** ich Zeit **habe**.*

### Adverbs that flip word order (verb 2nd, subject 3rd)
- **deshalb** — therefore  → *Tennis macht Spaß, **deshalb spiele ich** es oft.*
- **dann** — then  → *…, **dann gehe ich** ins Bett.*
- **außerdem** — furthermore  → *Außerdem **bin ich** sportlich.*

## 🆘 Rescue / filler phrases (for when you get stuck or interrupted)

| German | English |
|---|---|
| **Können Sie das bitte wiederholen?** | Could you repeat that please? |
| **Können Sie bitte langsamer sprechen?** | Could you speak more slowly please? |
| **Wie sagt man "X" auf Deutsch?** | How do you say "X" in German? |
| **Moment mal …** | One moment … |
| **Also …** | So / well … |
| **Naja …** | Well … |
| **Das ist eine gute Frage.** | That's a good question. |
| **Ich bin nicht sicher, aber ich glaube …** | I'm not sure, but I think … |
| **Wie meinen Sie das?** | What do you mean? |
| **Entschuldigung, ich habe das nicht verstanden.** | Sorry, I didn't understand. |

> 💡 If you forget a word, **don't stop** — describe it: *"Es ist ein Ding, das …"* (It's a thing that …) or *"Es ist wie …"* (It's like …).

---

## 🎓 Quick grammar reflexes

- **weil** → verb to the END.   *…, weil es **Spaß macht**.*
- **um … zu …** → "in order to …".   *Um gesund **zu sein**, esse ich Obst.*
- **gern → lieber → am liebsten**   (like → prefer → like most)
- **man kann …** → "you can …" + infinitive at the end.
- **ich möchte …** = polite "I would like …" (always sounds good)
