# 🔑 Answer Key — 01 Components & Circuit Symbols

| Q | Answer |
|---|--------|
| 1a | **Cell** (long line = +, short line = −) |
| 1b | **Lamp / bulb** |
| 1c | **Ammeter** |
| 1d | **Voltmeter** |
| 1e | **Fixed resistor** |
| 1f | **Variable resistor** (rheostat) |
| 1g | **Diode** |
| 1h | **LED** (light-emitting diode) |
| 1i | **Motor** |
| 1j | **LDR** (light-dependent resistor) — arrows pointing **in** mean light is coming in to hit it |
| 2 | Current: **amperes (A)**, **ammeter**. Voltage: **volts (V)**, **voltmeter**. Resistance: **ohms (Ω)**, **ohmmeter** (or calculated from V ÷ I). |
| 3a | **In series** with the bulb (so the same current passes through it). |
| 3b | **In parallel** with the resistor (connected across both ends). |
| 4 | (i) A **diode**. (ii) If connected the wrong way (reverse-biased), **no current flows** — the circuit acts as if there is a break. |
| 5a | The wires are **joined** (electrically connected). |
| 5b | The wires **cross but are not joined** — they are just passing over each other. |
| 6 | 1 → **C**, 2 → **A**, 3 → **B**, 4 → **D**, 5 → **E** |
| 7 | A **battery** (cells in series). Total voltage = 1.5 + 1.5 = **3.0 V**. |
| 8 | Conductors (any two): **copper, aluminium, gold, iron, steel, graphite**. Insulators (any two): **plastic, rubber, glass, wood, dry air, ceramic**. |
| 9 | Copper is a **good conductor** so it carries the current. Plastic is a **good insulator**, so the coating stops the current escaping — this protects us from electric shocks and stops wires short-circuiting if they touch each other. |
| 10 | A **cell** is a single energy source (one chemical unit). A **battery** is **two or more cells** joined together. |
| 11 | Conventional current: **+ to −** (round the outside of the cell). Electrons: **− to +** (the opposite direction). |
| 12 | One correct diagram: a single loop containing — cell-cell (battery), switch, lamp, ammeter, all in series. A voltmeter is drawn across the lamp (in parallel). All wires straight, right angles, junction dots where the voltmeter joins the main loop. Example: <br><br> `─┤├┤├──/──(×)──(A)──┐` <br> `             │     │` <br> `           ─(V)─   │` <br> `             │     │` <br> `             └─────┘` |
