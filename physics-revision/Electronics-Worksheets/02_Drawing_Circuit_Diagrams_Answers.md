# 🔑 Answer Key — 02 Drawing Circuit Diagrams

| Q | Answer |
|---|--------|
| 1 | **Parallel** — "side-by-side between the same two wires" describes two branches between the same two points. |
| 2 | **Series** — "one after the other in a single loop" describes a single-path circuit. |
| 3 | Single loop with: battery (two cells), switch, lamp, all in series. <br><br>`─┤├┤├──/──(×)──┐` <br>`│              │` <br>`└──────────────┘` <br><br>Marks for: ruler-straight wires, right-angle corners, correct cell + switch + lamp symbols, closed loop. |
| 4 | A single switch in the wire before the branches; then the circuit splits into two parallel branches, each containing one lamp; the branches rejoin and return to the cell. <br><br>`     +    −` <br>`    ─┤├──/──┬───────┐` <br>`    │       │       │` <br>`    │      (×)     (×)` <br>`    │       │       │` <br>`    └───────┴───────┘` <br><br>The switch turns **both** lamps on/off because all current goes through it. |
| 5 | Two parallel branches between the cell, but the switch is in **only one branch** (e.g. branch 1). When the switch is open, **only the lamp in that branch** goes out; the other lamp stays on. So the switch controls **just one** lamp. |
| 6 | Three mistakes: <br>**1.** A **voltmeter (V) has been placed in series**, in the main loop — it should be in parallel across the component you want to measure. <br>**2.** The **switch is drawn diagonally** — switches should be drawn with a clear horizontal "lever" lifted off the contact, not as a random diagonal line. <br>**3.** The **wire curves** at the bottom-right corner — wires must be straight with **right-angle** corners. (Also acceptable: the circuit is not closed properly / missing junction.) |
| 7 | Single loop with cell, ammeter (A) in series, resistor; voltmeter (V) drawn **across** the resistor (parallel to it). <br><br>`─┤├──(A)──┬──[ ]──┬──┐` <br>`│         │       │  │` <br>`│         └──(V)──┘  │` <br>`│                    │` <br>`└────────────────────┘` |
| 8 | An ammeter has **very low resistance**. Connecting it in parallel with the lamp creates a **short circuit** across the lamp: nearly all the current bypasses the lamp and rushes through the ammeter. The lamp barely lights, and the **large current** may **damage the ammeter** (and may blow the fuse). |
| 9 | A voltmeter has **very high resistance**. Putting it in series makes the **total resistance huge**, so almost **no current** flows in the circuit. The lamp will not light, and the voltmeter will simply read close to the full source voltage (because almost all of it drops across the voltmeter). |
| 10 | Battery (3 cells), then **ammeter in series in the main wire**, then a **switch**, then the wire splits into two parallel branches. Branch 1: one lamp. Branch 2: two lamps **in series**. Branches rejoin and return to the battery. <br><br>**a)** The ammeter reads the **total current** from the battery (because it is in the main wire before the split). <br>**b)** The cell voltage is shared **across each whole branch**. So the voltage across the single lamp (branch 1) equals the voltage across the **pair** of lamps in branch 2 — **not** across just one lamp in branch 2. (Each lamp in branch 2 only gets half the branch voltage, so they are dimmer than the lamp in branch 1.) |

### Marking tips
- **Q1–Q2**: 1 mark each.
- **Q3**: 4 marks — symbols correct (1), all four components present (1), closed loop (1), neat/ruler/right angles (1).
- **Q4**: 4 marks — two parallel branches (1), switch in main wire before split (1), both lamps in their own branches (1), neatness (1).
- **Q5**: 3 marks — diagram (2 — branches plus switch in one branch only), conclusion "only one lamp" (1).
- **Q6**: 3 marks (1 per mistake spotted).
- **Q7**: 4 marks — ammeter in series (1), voltmeter in parallel with resistor (1), correct symbols (1), neat closed loop (1).
- **Q8**: 3 marks — "low resistance" (1), "short-circuits the lamp / lamp doesn't light" (1), "may damage the ammeter / blow fuse" (1).
- **Q9**: 3 marks — "high resistance" (1), "very little current" (1), "lamp doesn't light / reads ≈ source voltage" (1).
- **Q10**: 4 marks — correct diagram (2), part (a) "total current" (1), part (b) "the pair of lamps" (1).

**Total available: ≈ 32 marks.**

### Common slip-ups to watch for
- Forgetting + and − on the cell.
- Drawing the long line as − (it's actually +).
- Forgetting the junction dot where the voltmeter meets the main wire.
- Curving wires round corners instead of using right angles.
- Mixing up "in series" and "in parallel" descriptions in word questions.
