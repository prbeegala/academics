### 🔑 Key Ideas

- A circuit diagram is a **simplified picture** of a circuit using **standard symbols**.
- Always use a **ruler** and **pencil**. Wires are **straight lines** with **right-angle (90°) corners**.
- The circuit must form a **complete loop** back to the cell (unless a switch is deliberately open).
- Mark **+ and −** on the cell, and put **arrows** on the wires showing conventional current (+ → −).
- **Ammeter** = in series (in the main loop). **Voltmeter** = in parallel (across the component).
- A **junction dot** at a wire crossing means the wires are joined. **No dot** means they just cross over.
- A circuit is **series** if there is **one loop** (one path for the current). It is **parallel** if components are on **separate branches** between the same two points.

### ✏️ Worked Example

**Q.** Draw a circuit diagram of: *a cell, a switch, two bulbs in series, and an ammeter in the loop.*

```
   +    −
  ─┤├──────/──────(×)──────(×)──────(A)──┐
  │                                       │
  └───────────────────────────────────────┘
```

Notes:
- One loop → series circuit.
- All four components are in the main loop, so the ammeter reads the same current as flows through each bulb.
