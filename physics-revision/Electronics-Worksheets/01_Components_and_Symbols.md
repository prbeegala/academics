### 🔑 Key Ideas

- Circuit diagrams use **standard symbols**, not pictures. You must be able to recognise and draw them.
- A **cell** is shown by one long line and one short line. The **long line is the positive (+) terminal**.
- A **battery** is just two or more cells joined together.
- An **ammeter (A)** measures **current** and goes **in series** with the component.
- A **voltmeter (V)** measures **voltage** and goes **in parallel** with the component.
- Memory hook: **A**mmeter → **A**long the wire; **V**oltmeter → a**V**oss the component.
- A **fixed resistor** is a plain rectangle. A **variable resistor** has an arrow through it.
- A **diode** lets current flow only **one way** (in the direction of the arrow/triangle).
- An **LED** is a diode that **emits light** — its symbol has two small arrows pointing away.
- A **fuse** is a safety device — it melts if the current is too high.
- A **junction dot** means two wires are joined. No dot = wires just cross over.

---

### ✏️ Worked Example

**Q.** A student wants to measure the voltage across a lamp and the current through it. Where should they put the meters?

**A.** The **ammeter** must be in **series** with the lamp (in the main loop) so the same current flows through it.
The **voltmeter** must be in **parallel** with the lamp (connected across it) so it reads the voltage between the two ends.

```
      +    −
    ─┤├──────(A)──────┐
    │                  │
    │              ┌──(×)──┐    ← lamp with voltmeter
    │              │       │       across it
    │              └──(V)──┘
    │                  │
    └──────────────────┘
```
