# Maths Daily Practice — Prompt Summary

## Context
- **Student:** Shrishti, Year 8, St Paul's Girls School, London
- **Goal:** Build maths confidence through daily practice
- **Format:** 20 minutes per day, at least 10 questions (we do 15)

---

## Prompt 1 — Initial Request
> Create a list of 10 questions on a daily basis covering: **Indices, Quadratic equations, Algebraic factorisation, Expansion of brackets, Surds, Geometry using algebra**. Display 15 questions to start, and adjust difficulty as needed.

**Result:** 15 sample questions displayed at moderate Y8 level.

---

## Prompt 2 — Generate Printable Sheets (Week 1)
> This difficulty is good for the first 5 days. Create printable files for each day. Create a folder called **Shrishti** under **Projects**, and inside Shrishti create a folder called **MathsDailyPractice**, then upload the files there.

**Result:** 5 daily sheets (Day 1–5) + Answer Key created in `Projects\Shrishti\MathsDailyPractice\`. Difficulty: moderate, appropriate for a strong Y8 student.

---

## Prompt 3 — Week 2 (Harder)
> Create Week 2 with greater difficulty.

**Result:** 5 daily sheets (Day 6–10) + Answer Key. Increased difficulty includes:
- Fractional/negative indices with variables
- "Show that" and proof questions
- Surd answers expected (not just integers)
- Real-world modelling (projectiles, 3D shapes)
- Multi-step geometry problems

---

## Prompt 4 — Week 3 (Same Difficulty + New Topics)
> Create Week 3 as well, keep the same level of difficulty — but also add **number sequences, nth term** and also **line equations**.

**Result:** 5 daily sheets (Day 11–15) + Answer Key. Same difficulty as Week 2, with 3 new topic areas:
- **Number sequences & nth term** (linear, quadratic, geometric sequences; arithmetic series)
- **Line equations** (y = mx + c, gradients, parallel/perpendicular lines, intersections)
- **Coordinate geometry** integrated into the geometry section

---

## Prompt 5 — Organise into Weekly Folders
> Create folders called Week 1, Week 2 etc. and move files accordingly for better organisation.

**Result:** Files reorganised into:
```
MathsDailyPractice/

└── Week3/   (Day 11–15 + AnswerKey_Week3)
```
