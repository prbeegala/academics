## 🎯 How to use this repo

1. **Pick a subject folder** for whatever you're working on.
2. **Read the folder's own `README.md`** (where present) for a study routine.
3. **Use the PDFs** under each `PDF/` subfolder for printing.
4. **Mark up Markdown files freely** — open in any editor (VS Code recommended).

## 🧭 Conventions

- Each subject folder is **self-contained** — notes, worksheets, answer keys all live together.
- Markdown is the source of truth; PDFs are generated for printing/offline use.
- Answer keys / mark schemes are kept in separate files (filenames usually end with `_Answers.md` or `_MarkScheme.md`).
