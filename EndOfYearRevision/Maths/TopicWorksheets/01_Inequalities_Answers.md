# 🔑 Answer Key — 01 Inequalities

| Q | Answer (brief working) |
|---|------------------------|
| 1 | **x < 7**. Working: x + 6 < 13 → x < 7 |
| 2 | **x ≤ 7**. Working: 3x − 5 ≤ 16 → 3x ≤ 21 → x ≤ 7 |
| 3 | **y ≥ −4**. Working: 4y + 9 ≥ −7 → 4y ≥ −16 → y ≥ −4 |
| 4 | **a > −4**. Working: 7 − 2a < 15 → −2a < 8. Divide by −2 and flip the sign → a > −4 |
| 5 | **−5 < x ≤ 6**. Working: −3 < x + 2 ≤ 8 → subtract 2 from all three parts → −5 < x ≤ 6 |
| 6 | **m ≥ 6**. Working: 2(3m − 1) ≥ 4m + 10 → 6m − 2 ≥ 4m + 10 → 2m ≥ 12 → m ≥ 6 |
| 7 | **n = −1, 0, 1, 2, 3**. Working: −2 ≤ 3n + 1 < 13 → −3 ≤ 3n < 12 → −1 ≤ n < 4 |
| 8 | **p = 4, 5, 6, 7, 8**. Working: 5 < 2p − 1 ≤ 15 → 6 < 2p ≤ 16 → 3 < p ≤ 8 |
| 9 | **4 + 2a ≤ 18, so a ≤ 7. Maximum = 7 activities.** Working: subtract 4 → 2a ≤ 14 → a ≤ 7 |
| 10 | **x ≤ −5**. Working: 11 − 3x ≥ 26 → −3x ≥ 15. Divide by −3 and flip the sign → x ≤ −5 |
| 11 | **x ≤ −2**. Working: multiply by 6: 2(2x + 1) − 3(x − 4) ≤ 12 → 4x + 2 − 3x + 12 ≤ 12 → x + 14 ≤ 12 → x ≤ −2. Check: x = −3 gives −5/3 − (−7/2) = 11/6 ≤ 2 ✓; x = 0 gives 1/3 − (−2) = 7/3 > 2 ✗ |
| 12 | **x = −2, −1, 0, 1, 2, 3, 4, 5, 6, 7**. Working: x² ≤ 49 means −7 ≤ x ≤ 7. Also x > −3, so integers are −2 to 7 |
