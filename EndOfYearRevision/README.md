## How to use the answer keys

1. Do the whole worksheet **without** looking at the answers.
2. Mark using the answer key.
3. For any question you got wrong, **redo it from scratch** before reading the working in the key.
4. If a topic feels shaky, ask for an extra worksheet on it.

Good luck — have fun with it! ✨
